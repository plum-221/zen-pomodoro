# 禅 · 番茄钟 · Zen Pomodoro

一个纯静态、禅意风格的番茄钟。专注时，一棵 SVG 植物会随着你完成的番茄慢慢生长——**把你的专注时间变成看得见的生命**。

打开就能用，无需登录，数据存在你自己的浏览器里。

## 功能

- 🍅 **番茄计时**：专注 / 短休 / 长休，默认 25 / 5 / 15 分钟，每 4 个专注后长休。全部时长可在设置里自定义。
- 🌱 **会生长的植物**：每完成一个番茄，植物长一节（种子→破土→抽茎→展叶→繁茂→含苞→绽放）；中途跳过它会蔫一下。纯 SVG 手绘，带弹性风力摇曳。
- ✅ **任务清单**：新增任务（名字 + 预计番茄数），可改名、删除、上下排序、打勾完成、选为「当前任务」；完成的番茄自动计到当前任务上。
- 🔔 **到点提醒**：柔和钟磬声（Web Audio 现场合成，无需音频文件）+ 页面提示 + 可选系统通知。
- ⏳ **关掉再回来能续上**：靠记录「目标结束时间戳」，重新打开时算出离开期间发生了什么，无缝接上或判定完成。
- 📱 **手机电脑自适应**，并支持 **PWA 安装**（离线也能开）。

## 本地运行

因为用到 service worker / 通知，直接双击 `index.html`（file://）部分功能会受限。建议起个本地服务器：

```bash
# 任选其一
python -m http.server 8000
npx serve
```

然后浏览器打开 `http://localhost:8000`。

## 部署到 GitHub Pages

1. 把本目录推到 GitHub 仓库。
2. 仓库 Settings → Pages → Source 选 `main` 分支根目录。
3. 稍等片刻得到网址，任何人打开即可使用。

## 已知边界（纯静态网页的限制）

- 数据只存在当前浏览器，换设备 / 换浏览器看不到，也无云同步。
- **网页完全关闭后不会提醒**——没有后台服务器。只要网页开着（哪怕在后台标签）就能提醒。
- PWA 安装只是让打开更快、可离线，同样不会在后台替你跑计时。

| 文件 | 作用 |

| `plant.js` | 植物 SVG 生成与生长、摇曳动画（可复用） |
