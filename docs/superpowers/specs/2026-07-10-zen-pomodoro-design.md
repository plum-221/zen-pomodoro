# 禅意番茄钟 · 设计文档

日期：2026-07-10 · 作者：Plum + Claude

## 一句话

一个纯静态、可放 GitHub Pages 让任何人打开即用的禅意番茄钟；核心趣味是**一棵随专注生长的 SVG 植物**，作为任务进程的见证。

## 关键约束（决定一切）

- **纯静态网页**：HTML/CSS/JS，无后端、无框架库。托管在 GitHub Pages，别人点开即用。
- 数据只存本地 `localStorage`（浏览器小抽屉）：刷新/关机不丢，但换设备/换浏览器看不到。无登录。
- 因为没有后台服务器，「关掉网页后还提醒」做不到（见「提醒」）。

## 视觉风格

- **自然禅（绿）**：苔绿 + 暖米渐变，圆润卡片，柔和光影。配色变量集中在 `:root`，换风格只改变量。
- 一套响应式布局，不做两版：
  - 电脑：计时器居中偏左，任务清单在右侧一栏。
  - 手机：计时器占上半屏，任务清单折到下面可滑动。
- 计时器为圆环进度 + `MM:SS`，植物在计时器旁静静生长。

## 功能

### 1. 计时器
- 三种模式：专注 / 短休 / 长休。默认 25 / 5 / 15 分钟，每 4 个专注后进入长休。
- 操作：开始、暂停、重置、跳过。
- **时长可自定义**：设置面板改三个时长 + 长休间隔，存 localStorage，可存预设。
- 倒计时用「记录目标结束时间戳」算法，而非傻数秒 → 后台标签被降速也不偷跑。

### 2. 任务（可编辑功能）
- 一条任务 = 任务名 + 预计番茄数。
- 顶部输入框「＋ 新任务」。任务可：改名、删除、上下移排序、打勾完成、选为「当前任务」。
- 专注完成一个番茄 → 当前任务已完成番茄数 +1。
- 全部存 localStorage。

### 3. 生长的植物 🌱
- SVG 矢量手绘 + 描边生长 + `easeOutBack` 弹性展叶 + 弹簧风力摇曳（原型见 `plant-demo.html`）。
- 每完成一个番茄，植物前进一个生长阶段：种子→破土→抽茎→展叶→繁茂→含苞→绽放。
- 中途放弃：植物「蔫一下」，给点心理成本。
- 植物成长进度随专注累计、存 localStorage。
- 可选延伸（v2，先不做）：每天一棵，长成后收进下方「小花园」。

### 4. 到点提醒（受纯静态约束，分层）
- **声音**：用 Web Audio 现场合成一记柔和钟磬声（无需音频文件），网页开着必响。
- **页面提示**：屏幕柔和变化 + 文字「该休息了 / 开始专注」。
- **系统通知**：Web Notifications，用户授权一次后，切到别的标签页也能弹。
- ⚠️ 做不到：网页**完全关闭**后提醒（无后端）。

### 5. 关掉浏览器回来能续上
- 开始专注即把「目标结束时间戳 + 当前模式 + 当前任务 + 植物阶段」写入 localStorage。
- 重新打开时用「现在时间」减一下重建状态：
  - 还没到点 → 无缝接着倒计时。
  - 已过点 → 判定该番茄已完成（植物照长），提示一句。
- 机制说明：网页关着时**没有在跑**，是回来时靠时间戳「一眼算出」离开期间发生了什么。

### 6. PWA（可安装、离线打开）
- `manifest.json` + service worker：可「安装」成桌面/手机图标，离线也能开。
- 注意：PWA 只是更快打开，关掉后同样不会在后台跑计时。

## 文件结构

- `index.html` — 主页面（结构 + 内联样式与脚本，或拆分见下）
- `styles.css` — 样式（配色变量集中，便于换风格）
- `app.js` — 逻辑：计时器、任务、植物、提醒、持久化、恢复
- `plant.js` — 植物 SVG 生成与生长/摇曳动画
- `manifest.json` + `sw.js` — PWA
- `README.md` — 说明 + GitHub Pages 使用方法

（也可全部塞进单 `index.html`；倾向拆分以便维护，最终以实现为准。）

## 不做（YAGNI）

- 账号/云同步/后端。
- 关闭网页后的真后台提醒。
- 程序化随机生成植物（L-system）——先用手绘分阶段。
- 花园/森林、统计报表、主题商店——留待 v2。

## 部署

- `git init` → GitHub 私有或公开仓库 → 开启 GitHub Pages → 得到人人可开的网址。
- 娱乐/可分享性质，倾向**公开**仓库。
